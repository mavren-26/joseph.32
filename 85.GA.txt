Problem 1: Activity Selection Problem
Problem Statement

You are given N activities with start and end times. You can only attend one activity at a time. Find the maximum number of activities you can attend.

Input:

N = number of activities

Two arrays start[] and end[] representing start and end times

Output:

Maximum number of activities you can attend

Example:

Input: start = [1, 3, 0, 5, 8, 5], end = [2, 4, 6, 7, 9, 9]
Output: 4
Explanation: You can attend activities at intervals (1,2), (3,4), (5,7), (8,9)
